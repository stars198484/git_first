В  Java  предоставляется  немало  классов  и  методов,  позволяющих  читать  и  записывать данные  в  файлы. Рассмотрим основные способы чтения и записи данных в файл. Для ввода-вывода данных в файлы чаще всего применяются  классы  FilelnputStream  и  FileOutputStream,  которые  создают  потоки  ввода-вывода байтов, связанные с файлами. Чтобы открыть файл для ввода-вывода данных, достаточно создать объект одного из этих классов, указав имя файла. Когда файл открыт для вывода, любой файл, существовавший ранее под тем же самым именем, уничтожается.
